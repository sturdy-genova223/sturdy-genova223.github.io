🎯 About this tool

Montagem-punch is a audio plugin designed to make your drums and percussion stand out in a mix. Transient shaping helps you adjust the initial impact of a sound. If your snare drum sounds weak, this plugin adds the punch you need. If a sound lasts too long, this plugin tightens it.

You use it as a VST3 plugin inside music software like Ableton Live, FL Studio, or Reaper. It runs on both Windows and macOS systems.

⚙️ System Requirements

To use this software, your computer must meet these basic hardware and software standards:

  • Operating System: Windows 10 or newer (or macOS 10.15+).
  • RAM: Minimum 4GB of memory.
  • CPU: A standard multi-core processor from the last five years.
  • Audio Software: A DAW (Digital Audio Workstation) that supports VST3 or Audio Unit plugins.

📥 Download and Installation

Follow these steps to add the plugin to your computer:

  1. Visit the official releases page.
  2. Locate the file ending in .exe for Windows.
  3. Download the installer to your computer.
  4. Double-click the file to open the installation window.
  5. Follow the on-screen instructions. The installer will place the VST3 file in your default plugin folder.

🎚️ How to use the plugin

Once you install the plugin, restart your music software so it can scan for new files. Load the plugin onto a track containing audio.

The interface is simple because it uses only one knob. Turn the knob to the right to increase the sharpness and punch of the audio. Turn it to the left to soften the sound and decrease the bite. Watch your volume levels, as increasing the punch adds energy to your track.

You can use this tool on individual drum tracks or on a whole group of drum sounds. It works well to help kicks and snares cut through dense mixes.

💡 Troubleshooting

If you cannot find the plugin in your music software, check your plugin manager settings. Ensure that your software looks at the folder where you installed the VST3 file. Usually, this is C:\Program Files\Common Files\VST3 on Windows machines.

If the sound distorts, turn the knob back until the signal stays within safe limits. This software does not change the color of your sound, only its shape.
